Access and Create the most powerful tools in modern physics

Tensor networks algorithms allow physicists to model and analyze complex systems of particles



All computations are performed with the DMRJtensor and TensorPACK Julia libraries

With this website, you can run state-of-the-art tensor network algorithms and even create your own!

What's included?

Go to the Run Algorithms page to do just that! Currently, DMRG can be run to find the ground state energy and correlations of a given system. A default Hamiltonain, such as the Heisenberg model, or a custom Hamiltonian can be supplied as the system.
( TRG, PEPS, and MERA coming soon )




Contract and decompose tensors effortlessly with the Build a Tensor Network page to design your own custom tensor network algorithms. Educate yourself on tensor network operations or test out new ideas for algorithms without the hassle of coding.
